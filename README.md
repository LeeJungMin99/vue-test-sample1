## Project Setup (Local)

### Clone the Repository

```sh
git clone https://github.com/LeeJungMin99/vue-test-sample1.git
```

### Install Packages

```sh
npm install
```

### Compile and Run Local Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
